# 🤖 Groq Модели для PCBuilderBot

## ✅ Рекомендуемые модели

### Основная модель
- **`openai/gpt-oss-120b`** - Рекомендуемая модель
  - Очень мощная модель от OpenAI
  - Отлично понимает технические задачи
  - Высокое качество объяснений
  - Оптимальна для анализа компонентов

### Альтернативные модели
- **`llama-3.1-8b-instant`** - Очень быстрая
  - Минимальная задержка ответа
  - Хорошо работает с JSON
  - Подходит для простых запросов
  
- **`llama-3.1-70b-versatile`** - Мощная модель
  - Более качественные ответы
  - Немного медленнее

## ⚠️ Устаревшие модели (НЕ ИСПОЛЬЗОВАТЬ)

- `llama3-70b-8192` - Декомиссирована
- `llama3-8b-8192` - Декомиссирована

## 🔧 Как сменить модель

В файле `.env`:
```env
AI_MODEL=openai/gpt-oss-120b
```

Или напрямую в `Bot/config/ai_config.py`:
```python
AI_MODEL = os.getenv("AI_MODEL", "openai/gpt-oss-120b")
```
