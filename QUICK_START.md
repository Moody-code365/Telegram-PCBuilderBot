# 🚪 Быстрый старт AI PCBuilderBot

## Что нужно сделать прямо сейчас:

### 1. Установить зависимости
```bash
pip install -r requirements.txt
```

### 2. Настроить .env файл
Добавьте в `.env`:
```env
TOKEN=your_telegram_bot_token
GROQ_API=your_groq_api_key
ENABLE_AI=true
AI_MODEL=llama-3.1-8b-instant
```

### 3. Запустить бота
```bash
python main.py
```

## 🔑 Где взять Groq API ключ?

1. Перейдите на [console.groq.com](https://console.groq.com)
2. Зарегистрируйтесь (бесплатно)
3. Создайте API ключ в разделе API Keys
4. Скопируйте ключ в `.env` файл

## ⚡ Быстрая проверка

Запустите тест:
```bash
python test_ai_integration.py
```

## 🎯 Результат

После настройки бот будет:
- 🤖 Использовать ИИ для умного подбора комплектующих
- 🔄 Автоматически переключаться на стандартную логику при проблемах
- 📱 Объяснять выбор компонентов (если ИИ доступен)
- 💰 Оптимально распределять бюджет

## 🆘 Проблемы?

**Если бот не запускается:**
1. Проверьте `.env` файл
2. Убедитесь что все зависимости установлены
3. Запустите `python test_ai_integration.py`

**Если ИИ не работает:**
- Бот продолжит работать в стандартном режиме
- Проверьте API ключ и интернет соединение
